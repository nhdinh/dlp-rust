**Step 1 — Create `dlp-server/tests/admin_audit_integration.rs`**

Create a new integration test file. The tests use `tokio` runtime, spawn a test server
instance backed by an in-memory SQLite database, and exercise the admin API endpoints.

The test module should follow this structure:

```rust

//!
//! Tests that policy CRUD and password-change operations emit audit events
//! with EventType::AdminAction, stored in the audit_events table.

use dlp_server::db::Database;

async fn test_policy_create_emits_admin_audit_event() { ... }
```

**Key patterns to follow:**
- Use an in-memory SQLite `Database::open(":memory:")` — avoids test isolation issues
- Create an admin user in the DB before each test (password bcrypt-hashed)
- Issue a JWT via the test setup, then attach `Authorization: Bearer <token>` to each request
- After each HTTP operation, query `audit_events` directly via SQLite (not the HTTP API)
  to verify the event was stored
- Assert on: `event_type = "ADMIN_ACTION"`, `action_attempted`, `resource_path`,
  `user_name`, `decision`, `agent_id`

**Test setup helper:**

```rust

```

Since `AppState` may have additional fields (siem, alert, etc.), use `Default::default()`
where possible and only seed what tests need. If `AppState` requires non-Default fields,
use `Arc::new(state)` with only the `db` field populated and `None` for optional relays.

**Step 2 — `test_policy_create_emits_admin_audit_event`**

2a. Start an in-memory server with a real `Database` and a pre-seeded admin user.
2b. Issue a JWT for the admin user (use `admin_auth::encode` with a test secret).
2c. `POST /admin/policies` with a test policy payload.
2d. `SELECT` from `audit_events` where `event_type = 'ADMIN_ACTION'` and
    `action_attempted = 'POLICY_CREATE'` and `resource_path LIKE 'policy:%'`.
2e. Assert exactly 1 row returned. Assert `user_name` matches the JWT subject,
    `classification = 'T3'`, `decision = 'ALLOW'`, `agent_id = 'server'`.

```rust
#[tokio::test]
async fn test_policy_create_emits_admin_audit_event() {
    let (state, db) = setup_test_app();
    let jwt = issue_test_jwt("admin");

    // POST /admin/policies
    let resp = test_request()
        .method(axum::http::Method::POST)
        .path("/admin/policies")
        .header("Authorization", format!("Bearer {jwt}"))
        .json(&serde_json::json!({
            "id": "test-policy-audit",
            "name": "Test Policy",
            "priority": 100,
            "conditions": [],
            "action": "DENY",
            "enabled": true
        }))
        .send(&state)
        .await;
    assert_eq!(resp.status(), 201);

    // Verify audit event
    let conn = db.conn().lock();
    let row: (String, String, String, String) = conn
        .query_row(
            "SELECT event_type, action_attempted, resource_path, user_name \
             FROM audit_events \
             WHERE action_attempted = 'POLICY_CREATE'",
            [],
            |row| Ok((row.get(0)?, row.get(1)?, row.get(2)?, row.get(3)?)),
        )
        .expect("audit event must exist");
    drop(conn);

    assert_eq!(row.0, "ADMIN_ACTION");
    assert_eq!(row.1, "POLICY_CREATE");
    assert!(row.2.starts_with("policy:"));
    assert_eq!(row.3, "admin");
}
```
