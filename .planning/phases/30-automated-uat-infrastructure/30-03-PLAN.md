   - Call `helpers::server::build_test_app()` to get router + pool.
   - Bind TCP listener to 127.0.0.1:0, get local address.
   - Spawn `axum::serve(listener, router)` in background tokio task.
   - Call `helpers::tui::build_test_app_with_mock_client(format!("http://{addr}"))`.
   - Return `(app, pool, addr)`.

3. **Test: `test_navigate_to_managed_origins`**:
   - Setup app.
   - `inject_key_sequence` with: Down, Down, Down, Enter (MainMenu -> DevicesMenu).
   - Assert `matches!(app.screen, Screen::DevicesMenu { selected: 0 })`.
   - `inject_key_sequence` with: Down, Enter (select "Managed Origins" at index 1).

   - Render with `render_to_buffer(&app, 80, 24)`.
   - Assert buffer content contains "Managed Origins".

4. **Test: `test_add_managed_origin`**:
   - Setup app, navigate to ManagedOriginList.
   - Inject 'a' (KeyCode::Char('a')).
   - Assert screen is `TextInput` with prompt containing "Origin URL pattern".
   - Type "https://company.sharepoint.com/*" by injecting KeyEvent for each char, then Enter.
   - Poll screen state with `tokio::time::sleep(Duration::from_millis(100)).await` between checks.
   - Assert screen becomes `ManagedOriginList` with `origins.len() == 1`.
   - Assert `origins[0]["origin"].as_str() == Some("https://company.sharepoint.com/*")`.
   - Render and assert buffer contains "company.sharepoint.com".

5. **Test: `test_remove_managed_origin`**:
   - Setup app, add one origin (reuse inline sequence or helper), navigate to ManagedOriginList.
   - Extract the origin id: `let id = origins[0]["id"].as_str().unwrap().to_string();`
   - Inject 'd'.
   - Assert screen is `Confirm` with message containing the origin URL.
   - Inject Enter (yes_selected defaults to true).
   - Poll until screen becomes `ManagedOriginList`.
   - Assert `origins.is_empty()`.

6. **KeyEvent construction** (same pattern as 30-02):

   let down = KeyEvent::new(KeyCode::Down, KeyModifiers::NONE);

   let char_d = KeyEvent::new(KeyCode::Char('d'), KeyModifiers::NONE);
   ```

   ```
