## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| spawn_blocking outside Tokio | tokio::task::spawn_blocking() panics when called from a std::thread with no reactor. The argument to block_on() is evaluated synchronously BEFORE block_on sets up any context. | advisory | In std::thread: call blocking functions directly. spawn_blocking is only needed inside async tasks on the Tokio executor. |
