  <behavior>
Tests must cover these observable behaviors (D-03):

**Publisher field:**
- `SourceApplication { field: Publisher, op: "eq", value: "Contoso" }` + ctx with source_app.publisher == "Contoso" → policy MATCHES → DENY
- `SourceApplication { field: Publisher, op: "eq", value: "Contoso" }` + ctx with source_app == None → condition FAILS CLOSED → does NOT match policy
- `SourceApplication { field: Publisher, op: "ne", value: "Contoso" }` + ctx with source_app.publisher == "Microsoft" → policy MATCHES
- `SourceApplication { field: Publisher, op: "ne", value: "Contoso" }` + ctx with source_app == None → FAILS CLOSED

**ImagePath field:**
- `SourceApplication { field: ImagePath, op: "contains", value: "Program Files" }` + ctx with image_path == r"C:\Program Files\App\app.exe" → MATCHES
- `SourceApplication { field: ImagePath, op: "contains", value: "Untrusted" }` + ctx with image_path == r"C:\TrustedApp.exe" → does NOT match
- `SourceApplication { field: ImagePath, op: "eq", value: r"C:\app.exe" }` + exact match → MATCHES
- `SourceApplication { field: ImagePath, op: "contains", value: ... }` + None identity → FAILS CLOSED

**TrustTier field:**
- `SourceApplication { field: TrustTier, op: "eq", value: "trusted" }` + trust_tier == Trusted → MATCHES
- `SourceApplication { field: TrustTier, op: "eq", value: "untrusted" }` + trust_tier == Trusted → does NOT match
- `SourceApplication { field: TrustTier, op: "ne", value: "trusted" }` + trust_tier == Untrusted → MATCHES
- `SourceApplication { field: TrustTier, op: "eq", value: "unknown" }` + trust_tier == Unknown → MATCHES

**DestinationApplication:**
- `DestinationApplication { field: Publisher, op: "eq", value: "Contoso" }` + ctx with dest_app.publisher == "Contoso" → MATCHES
- `DestinationApplication { field: Publisher, op: "eq", value: "Contoso" }` + ctx with dest_app == None → FAILS CLOSED

**Unsupported operator:**
- `SourceApplication { field: Publisher, op: "contains", value: "soft" }` → returns false (contains not supported for Publisher per D-03)

**Boolean mode interaction (policy-level):**
- PolicyMode::ALL with one SourceApplication condition + one Classification condition — both must match
- PolicyMode::ANY with one SourceApplication (None identity, fails closed) + one Classification (matches) — Classification match still triggers DENY
