- GET /admin/managed-origins -> 200 + Vec<{id, origin}>  (unauthenticated)
- POST /admin/managed-origins + JWT -> 200 + {id, origin}
- POST /admin/managed-origins + JWT with duplicate origin -> 409 Conflict
- POST /admin/managed-origins without JWT -> 401 Unauthorized
- DELETE /admin/managed-origins/{id} + JWT -> 204 No Content
- DELETE /admin/managed-origins/{nonexistent-id} + JWT -> 404 Not Found
