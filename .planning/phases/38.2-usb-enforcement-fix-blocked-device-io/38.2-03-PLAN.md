<objective>
Fix the drive-letter mislabel in `dlp-common/src/disk.rs` so that `find_drive_letter_for_instance_id` correctly correlates a disk's `instance_id` to its actual drive letter.

Purpose: The audit log shows SanDisk USBSTOR at drive_letter "C" when the user plugged at F:. The root cause is that `find_drive_letter_for_instance_id` ignores the `_instance_id` parameter entirely and returns the first unassigned fixed drive letter. This breaks both audit clarity and DACL targeting (the secondary enforcement layer depends on correct drive letters).
Output: Rewritten `find_drive_letter_for_instance_id` that properly maps instance_id to drive letter using Windows volume-to-disk APIs.

### From disk.rs (lines 772-809) — BUGGY find_drive_letter_for_instance_id

```rust
fn find_drive_letter_for_instance_id(
    _instance_id: &str,
    _already_found: &[DiskIdentity],
) -> Option<char> {
    let drives = unsafe { GetLogicalDrives() };
    if drives == 0 {
        return None;
    }

    for letter in 'A'..='Z' {
        let bit = 1u32 << (letter as u32 - 'A' as u32);
        if drives & bit == 0 {
            continue;
        }

        let path = format!("{}:\", letter);
        let wide: Vec<u16> = path.encode_utf16().chain(std::iter::once(0)).collect();

        let drive_type = unsafe { GetDriveTypeW(windows::core::PCWSTR(wide.as_ptr())) };
        if drive_type != DRIVE_FIXED {
            continue;
        }

        if std::path::Path::new(&format!("{}:\", letter)).exists() {
            let already_taken = _already_found
                .iter()
                .any(|d| d.drive_letter == Some(letter));
            if !already_taken {
                return Some(letter);
            }
        }
    }

    None
}
```

The `_instance_id` parameter is prefixed with underscore — it is NEVER USED. The function simply returns the first fixed drive letter that hasn't been assigned to a previously enumerated disk.

### From RESEARCH.md — recommended fix approach

Use `FindFirstVolume` / `FindNextVolume` to enumerate all volumes, then for each volume:
1. Get mount points (drive letters) via `GetVolumePathNamesForVolumeMountPointW`
2. Get disk extents via `IOCTL_VOLUME_GET_VOLUME_DISK_EXTENTS` on the volume handle
3. Match the disk number to the SetupDi `instance_id` via `IOCTL_STORAGE_GET_DEVICE_NUMBER` on `\\.\PhysicalDriveN`

Alternative simpler approach (recommended for this plan):
For each fixed drive letter, open `\\.\X:`, send `IOCTL_STORAGE_GET_DEVICE_NUMBER` to get the device number, then match against the device number obtained from the SetupDi enumeration path. The SetupDi `instance_id` can be correlated to a `PhysicalDriveN` via `CM_Get_Child` / `CM_Get_Sibling` walk or via `SetupDiEnumDeviceInterfaces` with `GUID_DEVINTERFACE_DISK`.

Even simpler: The `enumerate_fixed_disks_windows` function already iterates over SetupDi disk devices. For each device, we can:
1. Get the device instance ID (already done)
2. Open the device interface path for that instance_id
3. Send `IOCTL_STORAGE_GET_DEVICE_NUMBER` to get the device number
4. Enumerate all volume GUIDs via `FindFirstVolumeW` / `FindNextVolumeW`
5. For each volume, open it and send `IOCTL_VOLUME_GET_VOLUME_DISK_EXTENTS`
6. Match the volume's disk number to the disk's device number
7. If matched, get the drive letter via `GetVolumePathNamesForVolumeMountPointW`

However, the simplest PRACTICAL fix that fits within context budget:

Use `SetupDiEnumDeviceInterfaces` with `GUID_DEVINTERFACE_DISK` to get the device interface path for the given `instance_id`, then send `IOCTL_STORAGE_GET_DEVICE_NUMBER` to that handle to get the disk number. Then enumerate volumes and use `IOCTL_VOLUME_GET_VOLUME_DISK_EXTENTS` to find which volume(s) map to that disk number. Finally, get the drive letter from the volume's mount points.

### Windows APIs needed (from windows crate)

- `FindFirstVolumeW`, `FindNextVolumeW`, `FindVolumeClose` (in `windows::Win32::Storage::FileSystem`)
- `GetVolumePathNamesForVolumeMountPointW` (in `windows::Win32::Storage::FileSystem`)
- `IOCTL_VOLUME_GET_VOLUME_DISK_EXTENTS` (in `windows::Win32::System::Ioctl`)
- `VOLUME_DISK_EXTENTS`, `DISK_EXTENT` structs
- `IOCTL_STORAGE_GET_DEVICE_NUMBER` (already imported)
- `STORAGE_DEVICE_NUMBER` struct (already used)

### Critical: non-Windows fallback

The function is `#[cfg(windows)]`. On non-Windows, `enumerate_fixed_disks` returns an empty vector, so this function is never called.

    fn disk_number_for_instance_id(instance_id: &str) -> Result<u32, DiskError> {
        // Locate the device node by instance ID.
        let wide_id: Vec<u16> = instance_id
            .encode_utf16()
            .chain(std::iter::once(0))
            .collect();
        let mut devinst: u32 = 0;
        let cr = unsafe {
            CM_Locate_DevNodeW(
                &mut devinst,
                windows::core::PCWSTR(wide_id.as_ptr()),
                CM_LOCATE_DEVNODE_NORMAL,
            )
        };
        if cr.0 != 0 {
            return Err(DiskError::PnpWalkFailed(format!(
                "CM_Locate_DevNodeW failed for instance_id: cr=0x{:08X}",
                cr.0
            )));
        }

        // Get the device interface path for GUID_DEVINTERFACE_DISK.
        // Use SetupDiGetClassDevsW with DIGCF_DEVICEINTERFACE | DIGCF_PRESENT,
        // then enumerate interfaces and match by device instance ID.
        let hdev = unsafe {
            SetupDiGetClassDevsW(
                Some(&GUID_DEVINTERFACE_DISK),
                windows::core::PCWSTR::null(),
                None,
                DIGCF_DEVICEINTERFACE | DIGCF_PRESENT,
            )
        };
        let hdev = hdev.map_err(|e| DiskError::SetupDiFailed(format!("SetupDiGetClassDevsW: {e}")))?;

        let mut index: u32 = 0;
        let mut disk_number: Option<u32> = None;
        loop {
            let mut devinfo = SP_DEVINFO_DATA {
                cbSize: std::mem::size_of::<SP_DEVINFO_DATA>() as u32,
                ..Default::default()
            };
            if unsafe { SetupDiEnumDeviceInfo(hdev, index, &mut devinfo) }.is_err() {
                break;
            }

            let this_instance_id = read_instance_id(hdev, &devinfo)?;
            if this_instance_id == instance_id {
                // Found the matching device. Get its interface detail to open a handle.
                let mut interface_data = SP_DEVICE_INTERFACE_DATA {
                    cbSize: std::mem::size_of::<SP_DEVICE_INTERFACE_DATA>() as u32,
                    ..Default::default()
                };
                if unsafe {
                    SetupDiEnumDeviceInterfaces(
                        hdev,
                        Some(&devinfo),
                        &GUID_DEVINTERFACE_DISK,
                        0,
                        &mut interface_data,
                    )
                }
                .is_ok()
                {
                    // Get the required buffer size for the interface detail.
                    let mut required: u32 = 0;
                    let _ = unsafe {
                        SetupDiGetDeviceInterfaceDetailW(
                            hdev,
                            &interface_data,
                            None,
                            0,
                            Some(&mut required),
                            None,
                        )
                    };
                    if required > 0 {
                        let mut buf = vec![0u8; required as usize];
                        let detail = buf.as_mut_ptr() as *mut SP_DEVICE_INTERFACE_DETAIL_DATA_W;
                        // SAFETY: SP_DEVICE_INTERFACE_DETAIL_DATA_W starts with cbSize (u32) then DevicePath.
                        unsafe { (*detail).cbSize = std::mem::size_of::<SP_DEVICE_INTERFACE_DETAIL_DATA_W>() as u32 };
                        let ok = unsafe {
                            SetupDiGetDeviceInterfaceDetailW(
                                hdev,
                                &interface_data,
                                Some(detail),
                                required,
                                None,
                                None,
                            )
                        };
                        if ok.is_ok() {
                            // Extract the device path from the detail structure.
                            let path_wide: Vec<u16> = unsafe {
                                std::slice::from_raw_parts(
                                    (*detail).DevicePath.as_ptr(),
                                    (required as usize - std::mem::size_of::<u32>()) / 2,
                                )
                            }
                            .iter()
                            .copied()
                            .take_while(|&w| w != 0)
                            .collect();
                            let path = String::from_utf16_lossy(&path_wide);

                            // Open the device and query its disk number.
                            let wide_path: Vec<u16> = path.encode_utf16().chain(std::iter::once(0)).collect();
                            let handle = unsafe {
                                CreateFileW(
                                    windows::core::PCWSTR(wide_path.as_ptr()),
                                    0, // no access needed for IOCTL
                                    FILE_SHARE_READ | FILE_SHARE_WRITE,
                                    None,
                                    OPEN_EXISTING,
                                    FILE_FLAGS_AND_ATTRIBUTES(0),
                                    None,
                                )
                            };
                            if let Ok(h) = handle {
                                disk_number = query_disk_number_for_handle(h);
                                let _ = unsafe { CloseHandle(h) };
                            }
                        }
                    }
                }
                break;
            }
            index += 1;
            if index > 1024 {
                break;
            }
        }
        let _ = unsafe { SetupDiDestroyDeviceInfoList(hdev) };
        disk_number.ok_or_else(|| DiskError::SetupDiFailed("could not determine disk number for instance_id".to_string()))
    }

    fn query_disk_number_for_handle(handle: HANDLE) -> Option<u32> {
        let mut sdn: STORAGE_DEVICE_NUMBER = unsafe { std::mem::zeroed() };
        let mut returned: u32 = 0;
        let ok = unsafe {

                None,
                0,

                std::mem::size_of::<STORAGE_DEVICE_NUMBER>() as u32,
                Some(&mut returned),
                None,
            )

    **Step C: Find drive letter for disk number**
    ```rust
    #[cfg(windows)]
    fn find_drive_letter_for_disk_number(target_disk: u32) -> Option<char> {
        use windows::Win32::Storage::FileSystem::{
            FindFirstVolumeW, FindNextVolumeW, FindVolumeClose,
            GetVolumePathNamesForVolumeMountPointW,
        };

        let mut volume_buf = [0u16; 256];
        let find_handle = unsafe { FindFirstVolumeW(volume_buf.as_mut_slice()) };
        let find_handle = match find_handle {
            Ok(h) => h,
            Err(_) => return None,
        };

        let result = loop {
            let volume_path = String::from_utf16_lossy(
                &volume_buf.iter().copied().take_while(|&w| w != 0).collect::<Vec<u16>>(),
            );

            // Open the volume for IOCTL.
            let wide_vol: Vec<u16> = volume_path.encode_utf16().chain(std::iter::once(0)).collect();
            let vol_handle = unsafe {
                CreateFileW(
                    windows::core::PCWSTR(wide_vol.as_ptr()),
                    0,
                    FILE_SHARE_READ | FILE_SHARE_WRITE,
                    None,
                    OPEN_EXISTING,
                    FILE_FLAGS_AND_ATTRIBUTES(0),
                    None,
                )
            };

            if let Ok(h) = vol_handle {
                // Send IOCTL_VOLUME_GET_VOLUME_DISK_EXTENTS.
                let mut extents_buf = vec![0u8; 512];
                let mut returned: u32 = 0;
                let ok = unsafe {
                    DeviceIoControl(
                        h,
                        IOCTL_VOLUME_GET_VOLUME_DISK_EXTENTS,
                        None,
                        0,
                        Some(extents_buf.as_mut_ptr() as *mut std::ffi::c_void),
                        extents_buf.len() as u32,
                        Some(&mut returned),
                        None,
                    )
                };
                if ok.is_ok() && returned >= std::mem::size_of::<VOLUME_DISK_EXTENTS>() as u32 {
                    // VOLUME_DISK_EXTENTS layout:
                    //   DWORD NumberOfDiskExtents;  // offset 0
                    //   DWORD Padding;              // offset 4
                    //   DISK_EXTENT Extents[1];     // offset 8
                    // DISK_EXTENT layout:
                    //   DWORD DiskNumber;           // offset 0
                    //   LARGE_INTEGER StartingOffset; // offset 8
                    //   LONGLONG ExtentLength;      // offset 16
                    let num_extents = u32::from_le_bytes([
                        extents_buf[0], extents_buf[1], extents_buf[2], extents_buf[3],
                    ]);
                    for i in 0..num_extents {
                        let offset = 8 + (i as usize * 24); // sizeof(DISK_EXTENT) = 24
                        if offset + 4 <= extents_buf.len() {
                            let disk_num = u32::from_le_bytes([
                                extents_buf[offset],
                                extents_buf[offset + 1],
                                extents_buf[offset + 2],
                                extents_buf[offset + 3],
                            ]);
                            if disk_num == target_disk {
                                // Found the volume. Now get its mount points (drive letters).
                                let mut path_names_buf = [0u16; 1024];
                                let mut returned_paths: u32 = 0;
                                let ok = unsafe {
                                    GetVolumePathNamesForVolumeMountPointW(
                                        windows::core::PCWSTR(wide_vol.as_ptr()),
                                        Some(&mut path_names_buf),
                                        &mut returned_paths,
                                    )
                                };
                                let _ = unsafe { CloseHandle(h) };
                                if ok.is_ok() {
                                    // path_names_buf contains null-terminated strings, double-null terminated.
                                    let names: Vec<u16> = path_names_buf
                                        .iter()
                                        .copied()
                                        .take_while(|&w| w != 0)
                                        .collect();
                                    if !names.is_empty() {
                                        let path_str = String::from_utf16_lossy(&names);
                                        // Extract drive letter from "X:\" format.
                                        if let Some(letter) = path_str.chars().next() {
                                            if letter.is_ascii_alphabetic() {
                                                break Some(letter.to_ascii_uppercase());
                                            }
                                        }
                                    }
                                }
                                let _ = unsafe { CloseHandle(h) };
                                break None;
                            }
                        }
                    }
                }
                let _ = unsafe { CloseHandle(h) };
            }

            if !unsafe { FindNextVolumeW(find_handle, &mut volume_buf) }.as_bool() {
                break None;
            }
        };

        let _ = unsafe { FindVolumeClose(find_handle) };
        result
    }
    ```
