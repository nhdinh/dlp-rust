    2. Change the existing `std::mem::forget(tray)` line to store in a thread_local instead.

    3. Add public functions:
       ```rust
       /// Queue a tooltip string for application on the next Tick.
       ///
       /// Thread-safe: can be called from any thread. The actual `set_tooltip`
       /// call happens on the main thread via `apply_pending_tooltip`.
       pub fn update_tooltip(text: &str) {
           *PENDING_TOOLTIP.lock() = Some(text.to_owned());
       }

       /// Apply any queued tooltip update. Must be called from the main thread
       /// (where the TrayIcon lives). Called from the Tick arm in app.rs.
       pub fn apply_pending_tooltip() {
           let pending = PENDING_TOOLTIP.lock().take();
           if let Some(text) = pending {
               TRAY_ICON.with(|cell| {
                   if let Some(icon) = cell.borrow().as_ref() {
                       // set_tooltip accepts Option<&str>; None clears the tooltip
                       let _ = icon.set_tooltip(Some(text.as_str()));
                   }
               });
           }
       }
       ```

    **In `dlp-user-ui/src/ipc/pipe2.rs`:**

    1. Add the atomic state and accessor near the top of the file (after imports):
       ```rust
       use std::sync::atomic::{AtomicBool, Ordering};

       /// Tracks the last known Agent->Server connection state received via IPC.
       /// Initialized to false (unknown/disconnected) until the agent broadcasts.
       static AGENT_SERVER_CONNECTED: AtomicBool = AtomicBool::new(false);

       /// Returns the last known Agent->Server connection state.
       pub fn agent_server_connected() -> bool {
           AGENT_SERVER_CONNECTED.load(Ordering::Relaxed)
       }
       ```

    2. In the match arm that dispatches `Pipe2AgentMsg` variants, add:
       ```rust
       Pipe2AgentMsg::ServerConnected { connected } => {
           AGENT_SERVER_CONNECTED.store(connected, Ordering::Relaxed);
       }
       ```

    ```rust
        // In practice the static is process-wide; test verifies initial type behavior.
        assert!(!agent_server_connected() || true); // always passes; documents the default
    }
    ```
    (A stronger integration test is not possible without a live pipe; document the limitation.)
